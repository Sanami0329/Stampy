# Stampy（スタンピー）

## 概要
テキスト入力がない、スタンプだけでやりとりするコミュニケーションアプリです。

## 背景
祖父がテキスト入力を伴うメールやその他のチャットアプリをうまく使いこなせないため、<br>
そんな祖父でも使えるようなコミュニケーションアプリを作ってみようと思いました。

## 使用技術
- Python
- Flask
* Docker
* Docker Compose
* PostgreSQL
* HTML
* CSS
* JavaScript
* Tailwind CSS

## 特徴
- テキストを打ち込まないのはもちろんのこと、お年寄りでも操作しやすいように文字やコンテンツが全体的に大きめです。
- 直感的なUIはあえて使用せず、操作機能はすべて画面上に文字で表現されています。
- リアクションを送らなくても気まずくならないように、既読した時点で「見たよ」スタンプが自動で送信されます。
- 気分や感情を表すスタンプのほか、体調などに関わるスタンプも揃えています。

## 機能
* ユーザー登録、ログイン
* 今の気分（今日の体調）スタンプ編集
* フレンド検索、追加、削除
* グループ作成、メンバー追加、グループ離脱
* トーク作成、トーク履歴削除、グループトーク離脱

## 使い方
#### スタンプを送る
1. お好みのスタンプを選びます。
2. 選んだスタンプについて、［少し／普通／とても］の中から程度を選びます。※選んだ時点で送信されます。

#### 送られたスタンプやメディアに対してリアクションを送る
1. 「リアクションする」ボタンを押します。
2. お好みのリアクションボタンを選びます。※選んだ時点で送信されます。
